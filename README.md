# project-rock-paper-scissors


A console-based Rock Paper Scissors game built with vanilla JavaScript.

## About
This project was built to practice core JavaScript fundamentals including 
functions, scope, and conditionals.

## How to Play
1. Clone the repository
   git clone git@github.com:tjmax26/project-rock-paper-scissors.git
2. Open index.html in your browser
3. Open the browser console (F12)
4. The game runs for 5 rounds — enter rock, paper, or scissors when prompted
5. Your score vs the computer is tracked and a winner is declared at the end
